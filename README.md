# Calculadora-De-Estatistica

## Como rodar o site
- pip install flask
- pip install scipy
- python app.py

Calculadora para as aulas de estatistica

- Início da implementação dos novos cálculos passados em aula
- Correções Necessárias:
- Modal dos novos cálculos ficarem quando dados forem enviados
- Mostrar na tela o resultado das probabilidades
- Selecionar os botões de opções de contas de acordo com a conta
- Aparecer na tela os dados inseridos e os intervalos escolhidos

- "could not convert string to float: '3,33'" -> não dá para usar virgula
- Erro agrupamento discreto: Inserir só 1 dado no dados desordenados
- Arrumar a parte dos cards
- Quando trocar de conta sem clicar no botão Limpar, já executar a função limpar
- Botão final da dist normal
- Quando a pessoa insere 0 ele não aparece
- Alerta quando a pessoa não escolher intervalo

Opcional:
- Fazer ele lembrar do intervalo que a pessoa escolheu depois de calcular tudo
